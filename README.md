# Hemerton CLI

> Version: **v0.0.1**

Hemerton's command line interface.

## Development environment

### Requisites

- Python 3.7+
- Virtualenv
- GNU Make

### Setting up the project

```bash
# create a new virtualenv from Python 3.7
make setup-virtualenv
# activate the virtualenv
source venv/bin/activate
# complete the setup
make setup
```
